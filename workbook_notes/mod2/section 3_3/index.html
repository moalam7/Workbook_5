Very often, you will find you need to dynamically create HTML! There are several ways you can add/remove/replace nodes in the HTML doc!

appendChild() -- adds the node to the end of the list of children of a specified parent node

removeChild() -- removes child node from specified parent node

replaceChild() -- replaces child node in a specified parent node with a different nope

insertBefore() -- to work with sibling element of another element, inserts a node before the referenced node as a child of a specified parent node

no insertAfter() function, but with a little work, can code your own with helper func
